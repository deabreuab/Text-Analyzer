# Text-Analyzer
## Proyecto Text Analyzer Laboratoria
<p align="justify">Este proyecto consiste en un analizador de texto desarrollado en HTML, CSS y JavaScript. La aplicación permite a los usuarios ingresar texto y realiza diversas funciones de análisis, como contar palabras, caracteres, números, calcular la suma de los números, y más.
</p>

## Características
- Conteo de Palabras: Analiza y muestra la cantidad de palabras en el texto ingresado.
 <br/>
 
- Conteo de Caracteres: Calcula y muestra la cantidad total de caracteres en el texto.
 <br/>
 
- Conteo de Caracteres sin Espacios: Ofrece el recuento de caracteres excluyendo espacios y signos de puntuación.
 <br/>
 
- Conteo de Números: Identifica y muestra la cantidad de números presentes en el texto.
 <br/>
 
- Suma de Números: Calcula la suma total de los números encontrados en el texto.
 <br/>
 
- Promedio de Longitud de Palabras: Determina y muestra el promedio de longitud de palabras en el texto.

## Instalación
1. Clona este repositorio: git clone https://github.com/tuusuario/text-analyzer.git
2. Navega al directorio del proyecto: cd text-analyzer
3. Instala las dependencias: npm install
4. Abre el archivo index.html en tu navegador web.
5. ¡Listo, eso debería funcionar! :nerd_face:

## Uso
1. Ingresa el texto que deseas analizar en el área de texto.
2. Observa los resultados en la barra lateral derecha.
3. Utiliza el botón "Limpiar" para restablecer el área de texto y los resultados.
